# 概览
- 环境：生产 / 测试 / 本地。
- 影响范围：哪些用户或服务。
- 当前状态：已修复 / 仍在观察。

# 症状
- 具体报错、截图或日志片段。
- 指标异常。

# 复现步骤
1. 操作或脚本。
2. 预期 vs 实际结果。

# 根因分析
- 真正的触发原因。
- 为什么监控/测试没有提前发现。

# 解决方案
- 临时处置方案。
- 根因修复步骤。

# 预防措施
- 添加监控、报警、测试用例。
- 文档和培训更新。
